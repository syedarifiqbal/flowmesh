Distributed Systems Patterns
Rate Limiting
Sliding window per API key. 1,000 events/min default. Counters in Redis 1.
Idempotency
Deduplicate by eventId. Keys stored in Redis 2 (AOF). Survives service restarts.
Circuit Breaker
In Go Delivery. Opens after N failures. Fails fast to protect downstream services.
Exponential Backoff
Retry with doubling delay. Max 5 attempts per destination. Jitter to avoid thundering herd.
Dead Letter Queue
Failed deliveries land in RabbitMQ DLQ. Full visibility. One-click replay from dashboard.
Saga Pattern
Pipeline execution: coordinated rollback if a stage fails. No partial dirty state.
CAP Theorem
Ingestion = AP (never drop events). Delivery = CP (guarantee exactly-once).
Connection Pooling
PgBouncer sits in front of Postgres. All services connect to pooler, never directly to DB.
